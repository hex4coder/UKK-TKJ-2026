# Project Overview: UKK Teknik Jaringan Komputer (TKJ) 2026

This directory contains the official documents for the **Uji Kompetensi Keahlian (UKK)** — the Vocational Competency Test — for the **Teknik Komputer dan Jaringan (TKJ)** (Computer and Network Engineering) major for the 2025/2026 academic year in Indonesia.

The documents are organized into four main packages (Paket 1 to Paket 4), each representing a different exam scenario or technical focus.

## Directory Structure and Key Files

The files follow a specific naming convention based on the exam package and the type of document:

### File Type Suffixes:
*   **INV (Inventaris):** Lists the required equipment, tools, and materials for the specific exam package.
*   **KAP (Kisi-kisi / Kriteria):** Defines the assessment criteria, instrument grids, and the competencies being tested.
*   **PPsp (Pedoman Penskoran):** Provides the scoring guidelines and rubrics for the examiners.
*   **SPK (Soal Praktik Kejuruan):** The actual practical exam assignment given to the students, including technical requirements and network topologies.

## Usage

These documents are intended for:
1.  **Students:** To study the technical requirements and tasks (SPK) for their upcoming exams.
2.  **Teachers/Instructors:** To prepare the laboratory equipment (INV) and guide students through the criteria (KAP).
3.  **Assessors/Examiners:** To evaluate student performance using the scoring guidelines (PPsp).

## Technical Context
The exams typically involve practical implementation of network infrastructure:
- **Networking Hardware:** Configuring Routers (e.g., MikroTik), Managed/Unmanaged Switches, and client PCs.
- **Server Administration:** Setting up Linux/Windows servers for services like DNS, Web, and Monitoring (Zabbix/Prometheus).
- **Security:** Implementing Firewalls, VLANs, and hardening systems.
- **Monitoring:** Real-time network and server monitoring.
- **Physical Cabling:** Planning, cutting, crimping (e.g., Straight-Through T568B), and testing UTP cables (Cat 5e/6) with RJ-45 connectors.
- **IP Addressing:** Designing IP allocation, subnets, and routing (e.g., DHCP, Static IP).
